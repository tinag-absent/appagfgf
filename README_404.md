# 404エラーページについて

## 概要
海蝕機関のテーマに合わせたカスタム404エラーページを実装しました。

## 特徴

### 🎨 デザイン
- サイバーパンク風のダークテーマ
- グリッチエフェクト付きの404表示
- アニメーション効果（スキャンライン、パルス、グリッド）
- レスポンシブデザイン対応

### 🔧 機能
- ダッシュボードへの直接リンク
- トップページへの直接リンク
- システムステータス表示
- エラー詳細情報の表示
- ランダムなグリッチエフェクト

### 📱 対応環境
- デスクトップ
- タブレット
- モバイル

## セットアップ方法

### Apache サーバー
`.htaccess`ファイルが既に含まれているため、特別な設定は不要です。
ファイルをアップロードするだけで自動的に動作します。

**注意:** `.htaccess`ファイルが隠しファイルなので、FTPクライアントで「隠しファイルを表示」を有効にしてください。

### Nginx サーバー
`nginx.conf.example`を参考に、Nginxの設定ファイルに以下を追加してください:

```nginx
error_page 404 /404.html;
location = /404.html {
    internal;
}
```

詳細は`nginx.conf.example`ファイルを参照してください。

### ローカル開発環境
Live Serverなどのローカルサーバーを使用する場合、存在しないURLにアクセスすると404ページが表示されます。

## カスタマイズ

### メッセージの変更
`404.html`内の以下の部分を編集してカスタマイズできます:

```html
<h1 class="error-title">次元座標エラー</h1>
<p class="error-message">
  指定されたページは存在しないか、<br>
  次元の歪みにより消失した可能性があります。
</p>
```

### リンク先の変更
ボタンのリンク先を変更する場合:

```html
<a href="./dashboard.html" class="btn btn-primary">
  ダッシュボードへ
</a>
```

### スタイルの調整
`<style>`タグ内のCSSを編集して、色やアニメーションを調整できます。

## エフェクトの説明

### グリッチエフェクト
- 404の数字がランダムに揺れる効果
- タイトルテキストが一瞬乱れる効果
- 自動的にランダムなタイミングで発動

### スキャンライン
- ブラウン管モニターのような走査線
- 上から下へ継続的に移動

### グリッドアニメーション
- 背景のグリッドが斜めに移動
- サイバー空間を表現

### パルスエフェクト
- ステータスインジケーターの点滅
- システムが稼働中であることを表現

## トラブルシューティング

### 404ページが表示されない
1. `.htaccess`ファイルが正しくアップロードされているか確認
2. サーバーが`.htaccess`を有効にしているか確認（`AllowOverride All`が設定されているか）
3. Nginxの場合、設定ファイルに`error_page`ディレクティブを追加

### アニメーションが動かない
- JavaScriptが有効になっているか確認
- ブラウザのコンソールでエラーを確認

### スタイルが適用されない
- `css/style.css`が正しく読み込まれているか確認
- ブラウザのキャッシュをクリア
